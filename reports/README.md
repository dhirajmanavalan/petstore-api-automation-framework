# Petstore API Automation Framework

## Tech Stack
- Python
- PyTest
- Requests
- Logging
- CSV Handling

## Features
- CRUD operations for User, Pet, Store
- Bulk user creation
- CSV data logging
- Centralized API client
- Timestamped HTML reports
- Structured logging

## How to Run
pip install -r requirements.txt
pytest

## Project Structure
- utils/
- tests/
- logs/
- reports/
